Wordle Clone using Meteor JS

System Requirements

Bla bla

Project Setup

Install Meteor

npm install -g meteor

Create a basic project

meteor create --vue wordle-clone

If you receive any errors creating the project, check your console for any tips. In my case, I needed to run the following commands:

meteor npm install --save @babel/runtime
meteor npm install --save vue vue-meteor-tracker

Run the project.

meteor

Type http://localhost:3000 in your browser, and check if the app is loaded correctly.

Meteor creates a lot of files from the boilerplate project that we will not need. Start by removing the fixture.js file in the root folder, and then remove its import on main.js. Also, remove all files from collections folder, and, remove both files inside methods and publications. Finally, remove all components from the ui folder, and their imports on App.vue.

Client Development

Download bootstrap and add both bootstrap.min.css and bootstrap.min.css.map on imports folder, creating a new folder named bootstrap. Then, import it on main.js.

import "../imports/bootstrap/bootstrap.min.css";

Let's change the main colors of the application. On App.vue, change both background-color and color on the body entry in style.

body {
    padding: 10px;
    background: #121213;
    color: #d7dadc;
}

Let's shrink the application area. The quickiest way to do it is overring the value of max-width of the container inside the style. We only want to define the value when the container width is less than 576px. Start by adding the container class to the only div inside the application, and then, add the style code inside the style tag.

@media (min-width: 576px) {
    .container, .container-lg, .container-md, .container-sm, .container-xl, .container-xxl {
        max-width: 500px;
    }
}
